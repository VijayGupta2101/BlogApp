# 📝 BlogVerse

A modern and responsive **Blog Application** built with **Angular** for the frontend and **Node.js** for the backend. The application allows users to create, edit, publish, and manage blog posts through a clean and intuitive interface. It uses **Supabase** for authentication and database services, while **Tailwind CSS** provides a modern and responsive user experience.

---

## 🚀 Features

- 🔐 Secure User Authentication (Sign Up / Login / Logout)
- 👤 User Profile Management
- ✍️ Create, Edit, and Delete Blog Posts
- 📖 Read Published Blogs
- 🔍 Search Blogs
- 📱 Fully Responsive Design
- 🎨 Modern UI with Tailwind CSS
- ⚡ Fast Angular Frontend
- 🌐 RESTful API with Node.js & Express
- 🗄️ Supabase Database Integration
- 🔒 Protected Routes & Authentication
- 📂 Organized Project Structure

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/blog-app.git
```

### 2. Navigate to the Project

```bash
cd blog-app
```

### 3. Install Frontend Dependencies

```bash
cd frontend
npm install
```

### 4. Install Backend Dependencies

```bash
cd ../backend
npm install
```

---

## 🔑 Environment Variables

Create a `.env` file inside the backend folder.

```env
PORT=3000

SUPABASE_URL=your_supabase_url

SUPABASE_ANON_KEY=your_supabase_anon_key

SUPABASE_SERVICE_ROLE_KEY=your_service_role_key
```

---

## ▶️ Run the Application

### Start Backend

```bash
cd backend
npm start
```

### Start Angular Frontend

```bash
cd frontend
ng serve
```

Open your browser and visit

```
http://localhost:4200
```
